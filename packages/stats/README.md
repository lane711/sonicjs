# SonicJS Stats Collector

Self-hosted telemetry collection using SonicJS. Tracks anonymous installation metrics.

## Setup

1. Create D1 database:
   ```bash
   wrangler d1 create sonicjs-stats-db
   ```

2. Update `wrangler.toml` with your database ID

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run migrations:
   ```bash
   npm run db:migrate
   ```

5. Start dev server:
   ```bash
   npm run dev
   ```

## Collections

### installs
Tracks unique installations with anonymous IDs.

### events
Stores telemetry events (installation_started, installation_completed, installation_failed).

## API Endpoints

Auto-generated by SonicJS:

- `POST /v1/installs` - Create install record
- `POST /v1/events` - Track event

## Deployment

```bash
npm run deploy
```

## Privacy

- No PII collected
- Anonymous UUIDs only
- Public create access for telemetry
- Admin-only read/update/delete
